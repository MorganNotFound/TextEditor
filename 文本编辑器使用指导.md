# 使用方法
```
   （一）直接打字使用，若要保存，则选择另存为(A)
   （二）可以打开文件预览并修改，保存则到源目录，另存为则重新保存
   （三）点击格式-字体可以调节字体外观，但保存之后将以纯文本形式保存，如要保留格式，请自己使用Markdown语法，保存之后保存为md文件，作者将考虑添加Markdown模式
```
### Markdown格式示例：   
```
# 一级标题
## 二级标题
### 三级标题
*斜体*
**粗体**
***斜体加粗***
~删除线~

![图片链接](图片描述)
[链接地址](标题)
```
